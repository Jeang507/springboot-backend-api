# Spring Boot – Primeros Pasos

Este repositorio representa **mi primer acercamiento real al desarrollo backend con Spring Boot**.  
Aquí fue donde comencé a entender cómo funciona una API REST, cómo se organiza un proyecto Spring y cómo se conectan las distintas capas de una aplicación.

Aunque actualmente cuento con otros repositorios más avanzados, **este proyecto marca el inicio de mi camino en Spring Boot** y por eso decidí conservarlo como referencia.

## ¿Qué contiene este proyecto?

En este repositorio se implementan conceptos fundamentales de Spring Boot, tales como:

- Creación de una aplicación Spring Boot desde cero
- Uso de **@RestController** y mapeo de endpoints
- Arquitectura por capas:
  - Controller
  - Service
  - Model
- Operaciones CRUD en memoria
- Uso de distintos métodos HTTP:
  - GET
  - POST
  - PUT
  - PATCH
  - DELETE
- Lectura de datos desde un archivo JSON
- Manejo de rutas y parámetros (`@PathVariable`, `@RequestParam`)
- Uso de `ResponseEntity` para respuestas HTTP
- Inyección de dependencias

## Funcionalidades implementadas

### Estudiantes
- Listar estudiantes
- Buscar estudiante por correo
- Crear, actualizar y eliminar estudiantes
- Lectura de estudiantes desde un archivo JSON

### Clientes
- Listar clientes
- Buscar cliente por nombre de usuario
- Crear, actualizar y eliminar clientes
- Actualización parcial de datos

### Palíndromos
- Endpoint para validar si una frase es un palíndromo

## Tecnologías utilizadas

- Java
- Spring Boot
- Spring Web
- Jackson (JSON)
- Gradle
- Arquitectura REST

## Objetivo del repositorio

Este proyecto **no busca ser una aplicación final ni perfecta**, sino:

- Consolidar fundamentos de Spring Boot
- Servir como referencia de aprendizaje
- Mostrar mi progreso desde el inicio
- Recordar cómo estructuré mis primeros proyectos backend

Los repositorios posteriores contienen implementaciones más completas y mejores prácticas, pero **este es el punto de partida**.
